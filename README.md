# PhotonCNN
Repository for optimizing the flashgg photonID with a CNN 

## Setup
To setup:

    cmsrel CMSSW_10_5_0
    cd CMSSW_10_5_0/src
    cmsenv 

Clone via ssh:

    git clone git@github.com:atishelmanch/PhotonCNN.git

or https:

    git clone https://github.com/atishelmanch/PhotonCNN.git
    cd PhotonCNN 
    
To Run:

    cmsRun PhotonCNN_cfg.py
